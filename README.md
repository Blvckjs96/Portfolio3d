# 3D Portfolio Website

A modern, interactive 3D portfolio website built with React, Three.js, and Framer Motion. This portfolio features stunning 3D animations, smooth transitions, and a responsive design to showcase projects, experiences, and skills in an engaging way.

## Introduction

This is a personal portfolio website that combines modern web technologies with 3D graphics to create an immersive user experience. The site features interactive 3D elements, smooth animations, and a clean, professional design that adapts seamlessly across all devices.

## Features

- **Interactive 3D Graphics**: Powered by Three.js and React Three Fiber for stunning 3D visualizations
- **Smooth Animations**: Framer Motion for fluid page transitions and interactive elements
- **Responsive Design**: Fully responsive layout using Tailwind CSS
- **3D Globe**: Interactive globe component using COBE
- **Contact Form**: Integrated EmailJS for direct contact functionality
- **Project Showcase**: Dynamic project gallery with detailed views
- **Experience Timeline**: Visual timeline of professional experiences
- **Testimonials Section**: Client and colleague testimonials
- **Optimized Performance**: Built with Vite for lightning-fast development and production builds

## Tech Stack

### Core Technologies
- **React 19.1.1** - Latest React with improved performance and features
- **Vite 7.1.2** - Next-generation frontend tooling for blazing-fast builds
- **Three.js 0.179.1** - 3D graphics library for WebGL rendering

### 3D & Animation
- **@react-three/fiber** - React renderer for Three.js
- **@react-three/drei** - Useful helpers and abstractions for React Three Fiber
- **Framer Motion 12.23.12** - Production-ready animation library
- **COBE** - Lightweight 5kb WebGL globe library

### Styling & UI
- **Tailwind CSS 4.1.12** - Utility-first CSS framework
- **tailwind-merge** - Utility for merging Tailwind CSS classes

### Additional Features
- **EmailJS** - Email service for contact form functionality
- **react-responsive** - Media queries for responsive components
- **PapaParse** - CSV parsing library
- **Math.js** - Extended math library for JavaScript

### Development Tools
- **ESLint** - Code linting and quality assurance
- **PostCSS** - CSS transformation and optimization

## Prerequisites

Before running this project, make sure you have:
- **Node.js** (version 16 or higher)
